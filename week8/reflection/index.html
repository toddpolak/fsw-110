Using JavaScript to interact with an applicaton is important because it improves the user experience. It makes things more dynamic. Not having to continuously refresh the page is a big factor in allowing the user to make changes to a somewhat complex form or application. Being able to see their changes reflect immediately, knowing their progress in "real time" during an online application/form is what makes an application more useful to the user.

Event listeners make web page interaction a lot more easier in my opinion. Also the ability to manulipate the DOM with event listeners is a lot cleaner and a better user experience than the original "form post/get" methods. I felt suprised as to how easy it was actually to manipulate CSS and apply it to a web page via JavaScript. Compatibility issues can also be handled via JavaScript which makes development to support different platforms a lot easier.

My biggest accomplishment in this course was gaining a solid understanding of event listeners, manipulating the DOM, adding dynamic CSS to a web page and applying these concepts to real world applications.
